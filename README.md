Customer Behavior Analysis Project

Overview
This project aims to analyze customer shopping trends using data from 3,900 purchases across multiple US franchises. 
The goal is to uncover insights into spending patterns, customer segments, product preferences, and subscription behaviors to support strategic business decisions. 
The analysis pipeline includes data cleaning in Python, querying in SQL Server, and building an interactive dashboard in Power BI.

Data Description

The dataset contains 3,900 rows and 18 columns, with key features:

- Customer Demographics: Age, Gender, Location, Subscription Status
- Purchase Details: Item Purchased, Category, Purchase Amount, Size, Color, Season
- Shopping Trends: Review Rating, Discount Applied, Promo Code Usage, Previous Purchases, Payment Method, Shipping Type, Frequency of Purchases

A CSV file (`customer_behavior.csv`) was used for analysis.

Tools Used
- Python: Data cleaning, feature engineering, and exporting clean dataset.
- SQL Server Management Studio: Answering specific business questions via queries.
- Power BI: Creating an interactive dashboard for visual insights.
